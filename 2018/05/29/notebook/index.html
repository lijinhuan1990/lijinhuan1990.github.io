HT-seq软件

输入为sam格式的文件，如果是paired-end数据必须按照reads名称排序（sort by name）。可采用samtools先对bam文件（tophat2的输出结果为bam）排序，再转换为sam。

1.主要参数

-m 参数
计数模型，统计reads的时候对一些比较特殊的reads定义是否计入。包括：默认的union和intersection-strict、 intersection-nonempty

-s 参数
reads是否匹配到同一条链上，默认：yes，可以设置no 、 reverse

-t 参数
feature type 在gtf或者gff文件中，外显子为最小的定义单位，对基因计数，只需要将包含的外显子计数相加即可。 默认：exon

-i参数
最终的计数单位，一般为基因。 默认为：gene_id 也可以设置转录本，但由于模型问题，计数效果不佳。

-o参数
输出所有alignment的reads到一个sam文件中。可以不设置。

2.备注

可以设置转录本，但由于模型问题，计数效果不佳。

stringtie fpkm 转readcount脚本

之前主流分析软件：tophat2-cufflink,16年推出hisat2-stringtie-Ballgown后，rna分析流程update。hisat2-stringtie-Ballgown这一set，实测有两个问题：1.stringtie定量结果与cufflink相比，同一个转录本fpkm量较高；2.Ballgown只能对具有重复的样本进行差异分析，往往差异显著的基因较少，目标基因并为出现在差异显著组中，并且输入文件为fpkm。若进行差异软件更换， 一些差异分析软件输入文件为readcount值，因此需要将fpkm转换为readcount。官方推出转换脚本,可进行转换。

###图片相似度识别方法

下载安装win版Image Comparer ，安装完成后，打开软件。点击”工具“，选择“比较向导”，选择在“在一组图片”，选择存放有比较图片的文件夹，点击“类似的图像”，选择存放结果的位置，点击“处理”。即可得到相似度。
注意：文件夹中可以存放多个图片，将会两两比较。如果图片较为复杂，软件会报错。
获得相似度后，可以点击“对图像”，点击“高亮差异”便会显示出差异位点。

注意：实测感觉不大精准。有30天的试用期。

下载dup detector压缩包，运行DupDetector.exe，选择“Get data”,点击“Build”，点击“Browse”，选择存放有图片的文件夹，点击“OK”，点击“View dups”,选择Dup if里面的参数为99%，点击“Find dups”，便可获得本次分析结果。点击“View dups”，便可查看具体的分析结果。

注意：实测，感觉99%以上可以匹配的图片可能存在相似性，但是同样觉得比对结果差强人意。
