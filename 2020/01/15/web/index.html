    多个配对样本的差异分析 针对DESEQ2

    ####1.问题描述：如果拿到多个病人的癌和癌旁组织如何进行差异分析？####
    利用DESEQ2进行此类数据的分析时，在样本信息中应当有2列，分别是patientID(id1,id2,id3..) 和 treatment(control,case) 应当写成：design(dds) = ~ patientID + treatment
    注：patientID放在前面，treatment放在后面，表示主要研究的是treatment间的差异(control,case间),将各患者间的差异考虑为协变量。

    ####2.问题描述：如果拿到多个病人的癌和癌旁组织,并且每个病人有癌症分期或者age、sex信息需要考虑，如何进行差异分析？####
    利用DESEQ2进行此类数据的分析时，可以写成：design(dds) = ~ patientID + Phenotype +age + sex + treatment
    注：如果因素(斜变量)太多，软件可能会报错。
    注：如果根据数据或者文献报道，临床经验发现sex与treatment相关，利用DESEQ2进行此类数据的分析时，可以写成design(dds) = ~ patientID + Phenotype +age + sex + sex:treatment
