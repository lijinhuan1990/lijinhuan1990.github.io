exec介绍

可执行复杂的语句，python3内置函数

eval介绍

仅可执行字符串表达式

divmod介绍

对两个数字进行整除，并返回商和余数

divmod(7, 2)返回7%2的商和余数(3,1)

compile介绍

compile可将一个字符串编译为字节代码。可以和exec、eval联用，进行语句的执行

filter介绍

filter() 函数用于过滤序列，过滤掉不符合条件的元素。
filter(function, iterable)

hash介绍

hash() 用于获取取一个变量的哈希值，可用于校验变量在传输前后是否发生了变化。

repr介绍

repr()可返回一个对象的 string 格式

“‘RUNOOB’”

slice介绍

slice进行切片

slice(5) 对前5个进行切片
slice(0,4) 对0到4进行切片

import 介绍

import用于动态加载类和函数

pdb介绍

pdb 用于调试python 代码

非侵入式方法

1.启动pdb，进行调试

2.输入命令l 进行查看代码
3.输入命令n单步执行代码
4.逐行执行代码比较耗时，可以在特定行前面设置断点，然后运行命令c，可直接运行代码至断点处。运行命令b进行断点设置，如 b 110 即：在110行设置断点。
5.cl删除断点
6.执行命令a查看函数参数
7.p xxx 打印变量xxx的变量值
8.q 退出pdb模式
